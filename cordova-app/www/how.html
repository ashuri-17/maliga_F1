Curse MindManual

Game Overview

Welcome to Curse Mind! This is a word-guessing game with a spooky Halloween theme. You'll need to guess letters to reveal hidden words before you run out of lives. The game features different difficulty levels, hints, and a scoring system.

How to Play

1. Select a Difficulty Level

The game has four difficulty levels:

Easy

8 lives

Simple words

Medium

6 lives

Moderate words

Hard

3 lives

Challenging words

Extreme

1 life

Very difficult words

2. Guess Letters

  • Type a letter into the input box and press Enter or click Submit.
  • If the letter is in the word, it will appear in the correct position(s).
  • If the letter is wrong, you lose a life, and the incorrect letter is displayed.

3. Use the Hint

Each word comes with a hint to help you guess.

4. Complete Words to Progress

  • Each level has multiple words to guess.
  • Successfully guess all words to win!

5. Scoring System

  • Correct Guess: +10 points per correct letter.
  • Word Completion Bonus:
    • Easy: +100 points
    • Medium: +200 points
    • Hard: +300 points
    • Extreme: +500 points
  • Lives Bonus: Extra points for remaining lives.
  • Wrong Guess Penalty: -5 points.

6. Game Over

  • If you run out of lives, the game ends.
  • You can restart or return to the main menu.

Controls

  • Keyboard: Type letters and press Enter to submit.
  • Mouse: Click the Submit button after typing a guess.
  • Navigation:
    • Home Button: Returns to the main menu.
    • Settings Button: Adjust game options.

Tips & Strategies

  • Start with vowels (A, E, I, O, U).
  • Use the hint wisely.
  • Avoid repeating wrong letters.
  • Higher difficulties give more points but are riskier!

Game Features

Multiple Difficulty Levels
Halloween-Themed Words & Hints
Dynamic Scoring System
Progress Tracker
Responsive Design

Good luck, and have a spooky time! 🎃👻💀
